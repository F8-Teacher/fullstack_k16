## JavaScript

- Cú pháp (Ngôn ngữ lập trình nào cũng thế): biến, câu điều kiện, vòng lặp, hàm,...
- Kiểu dữ liệu:

* String: Nhớ các hàm xử lý cơ bản (Tra cứu), nguyên tắc xử lý chuỗi (Cắt 3 đoạn)
* Array: Học phương thức, cách hoạt động, tham chiếu
* Object: Học phương thức, cách hoạt động, tham chiếu
* ES6:

- Destructuring
- Spread
- Class
- Module: import, export

- DOM:

* Hiểu cách xử lý DOM => Tương tác vào object do trình duyệt xây dựng sẵn --> Tự động thay đổi HTML
* Tìm hiểu các sự kiện phổ biến (click, submit, mouse...)

- Bất đồng bộ

* Promise
* Async/Await

- Fetch API

* Cách client gọi dữ liệu từ server thông qua hàm fetch, axios
* Nhớ flow của auth

- Storage: Phân biệt 3 cách lưu trữ

- Package (npm): Nhớ các câu lệnh cài
